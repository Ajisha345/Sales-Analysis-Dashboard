# Sales-Analysis-Dashboard

## Overview
This project is a Power BI dashboard to analyze sales, profit, and regional performance.

## Tools Used
Power BI, Excel

## Key Insights
- Technology category has highest sales  
- Some categories have low profit  
- Sales trend changes over time  
- Different regions show different performance  

## Features
- Sales vs profit analysis  
- Category performance  
- Regional analysis  
- Time-based trends
